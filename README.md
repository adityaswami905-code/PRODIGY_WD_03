# Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe web game** built using **HTML, CSS, and JavaScript**.  
Players take turns placing X and O, and the game highlights the winning pattern.

## Features
- Two-player mode  
- Win and draw detection  
- Animated winning line  
- Score tracking  
- Responsive design
